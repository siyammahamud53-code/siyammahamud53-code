# 🚀 Siam's Universal AI Assistant Server

এটি সিয়ামের একটি অত্যন্ত শক্তিশালী এবং কাস্টমাইজড এআই ব্যাকএন্ড সার্ভার। জেমিনি এআই (Gemini API), ভয়েস ইঞ্জিন এবং ডিভাইস অটোমেশনের সমন্বয়ে তৈরি এই প্রজেক্টটি প্রায় ৪০টিরও বেশি ইউনিভার্সাল ধাপ ও মোড অনুসরণ করে কাজ করতে সক্ষম।

---

## 🛠️ প্রধান ফিচারসমূহ (Core Features)

* **এআই ব্রেন:** Google Gemini API-এর মাধ্যমে রিয়েল-টাইম ডিসিশন মেকিং।
* **ভয়েস কন্ট্রোল:** স্পিচ-টু-টেক্সট এবং টিটিএস (TTS) ইঞ্জিনের ব্যাকএন্ড সাপোর্ট।
* **ডিভাইস অটোমেশন:** মোবাইল ও গেম কন্ট্রোলের জন্য বিশেষ স্ক্রিপ্ট কানেক্টিভিটি।
* **ক্লাউড রেডি:** রেন্ডার (Render) এবং ডকার (Docker) ফ্রেন্ডলি আর্কিটেকচার।
